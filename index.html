Financial Formula: ROI

We use Return on Investment (ROI) to evaluate the efficiency of an investment.

$$ \text{ROI} = \frac{\text{Net Profit}}{\text{Cost of Investment}} \times 100\% $$

Risk Analysis Script

Python code used for calculating Value at Risk (VaR).


import numpy as np
from scipy.stats import norm

def calculate_var(portfolio_value, mu, sigma, confidence_level=0.95):
  z_score = norm.ppf(1 - confidence_level)
  var = portfolio_value * (mu + z_score * sigma)
  return abs(var)

value = 1_000_000
daily_return_mean = 0.0005
daily_return_std = 0.01

var_95 = calculate_var(value, daily_return_mean, daily_return_std)
print(f"95% Daily VaR: ${var_95:,.2f}")
